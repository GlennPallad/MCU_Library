# Usage
This file converts a 32bit number to a ASCII string.
For example:
`0xFFFF FFFF`->`"0b1111 1111 1111 1111 1111 1111"`

## Configuration
Configure your platforms in *num_to_ascii.h*.
* Currently supported:
`GCC_PLATFORMS`
`STM32_PLATFORMS`
